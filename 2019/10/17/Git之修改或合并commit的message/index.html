这里的a4e0c875cb6361635d57a8b3df56e0646cd2a727是你要修改的 commit 的父 commit 的 id。

我们可以看到第一行的pick 6bc1644 update index.htm to git.htm, 假如们再这个父 commit 之后还有多次 commit，这里会有多个 commit。

这里我们是要修改6bc1644这个 commit 的 message, 我们使用到下面提示中的reword, 意思是我还需要这个 commit，但是需要修改 message。 默认的pick就是这个 commit 不做任何变化。
