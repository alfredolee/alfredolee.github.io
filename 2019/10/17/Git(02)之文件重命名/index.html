Git(02)之文件重命名

上一节我们完成了一个完整的 commit，此时呢，我不想让他叫做index.html了，下面就来看看如何改名字。

1. 手动改名字，然后 git add/rm …

执行 git rm index.html

执行 git add git.html

可以看到，git 非常智能的知道了我们是在重新命名。但是呢，重新命名一个文件是在太繁琐了。这时候git mv就来了。

2. git mv …

此时，我们使用git reset --hard把暂存区所有的修改重置。之后再执行git mv

最后，别忘了git commit我们的修改。

我们可以看到, git mv index.html git.html这一条命令的效果和之前三条效果是一样的。至此，使用 git 重新命名文件也就完成了。
