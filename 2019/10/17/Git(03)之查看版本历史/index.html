Git(03)之查看版本历史

之前的两小节我们分别完成了两次提交，那么我们现在怎么查看之前的提交历史呢？git 给我们提供了git log命令：

1. git log

2. git log –oneline

2. git log –oneline

3. git log –all

-all, 是指查看所有分支的 commit 历史。我们使用git checkout -b temp a4e0c87基于第一次提交新建一个分支。此时，使用git log --all。

当然有时候我们不想看到所有的 commit, 我们可以使用git log -n1来查看最近一次 commit。

我们可以看到，在 temp 分支下，我们也看到了 master 分支下的 commit。但是感觉不够明了。

4. git log –all –graph

--graph, 相比之前，添加了一些图形化的标识使得表现更明了。

我们在temp分支，做出一个修改，形成一个 commit。在此 commit 的基础上，新建temp2分支。

我们可以看到，最左侧多了一个图形线，并且master分支是落后temp和temp2分支一个 commit, 上面图标很好的表现出来了。

5. 总结

  • git log –oneline 查看单行的简洁历史。
  • git log –all 查看所有分支的历史
  • git log –all –graph 查看图形化的 log 地址
  • git log –oneline -n4 查看最近的四条简洁历史。
  • git log –oneline –all -n4 –graph 查看所有分支最近 4 条单行的图形化历史。
  • git help –web log 跳转到 git log 的帮助文档网页
